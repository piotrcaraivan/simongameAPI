Правила игры Simon Memory Game

Simon Memory Game — это игра, в которой ваша задача — запомнить последовательность цветов, показанную компьютером, и воспроизвести её.

  1. Нажмите кнопку запуска, чтобы начать игру.
  2. Обратите внимание на цветовую последовательность, отображаемую на экране.
  3. Воспроизведите последовательность, нажимая соответствующие цветные кнопки.
  4. Каждый раунд добавляет новый цвет, усложняя игру.
  5. Игра завершится, если вы ошибётесь в последовательности.

Попробуйте свои силы и посмотрите, насколько далеко вы сможете продвинуться!
